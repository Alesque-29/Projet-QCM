# Projet-QCM

Réalisation d'une application utilisée dans un centre de formation fictif.  
Objectifs: permettre à des formateurs de consulter la liste de leurs élèves, et à des candidats de consulter la listes des épreuves qu'ils vont devoir passer.  
Authentification requise, mise en session utilisateur, cloisonnement entre une session de type formateur et de type candidat.  
Utilisation de Java EE, de SQL Server, de JDBC et d'une Factory.
